# AI-Powered Video & Meeting Assistant with RAG

A production-ready AI assistant built in Python to automate meeting workflows and video analytics. This system ingests multimedia inputs, generates multilingual transcriptions, extracts actionable business insights, and supports interactive contextual Q&A using Retrieval-Augmented Generation (RAG).

## 🚀 Key Features
- **Dynamic Ingestion:** Processes YouTube URLs and local audio/video file uploads.
- **Hybrid Transcription:** Utilizes local OpenAI Whisper for optimized English and Servam AI for high-accuracy Hindi speech-to-text workflows.
- **Workflow Automation:** Leverages LangChain LCEL to automatically extract structured summaries, key decisions, and track actionable items.
- **Contextual Q&A:** Implements a vector database index using ChromaDB and HuggingFace embeddings for localized semantic search.
- **AI Security:** Features built-in LLM gateway controls including PII masking and prompt injection blocking.

## 🛠️ Tech Stack
- **Languages:** Python
- **Orchestration:** LangChain (v1), LangGraph
- **Vector DB:** ChromaDB
- **LLMs:** Mistral AI, OpenAI Whisper, Servam AI
- **UI Framework:** Streamlit
- **Package Manager:** UV
